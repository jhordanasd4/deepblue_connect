# Imágenes del Mercado Submarino

Coloca aquí las imágenes que se usarán en el Mercado.

Nombres de archivo esperados (puedes usar .jpg o .webp):
- coral.jpg  → se referenciará como /img/coral.jpg
- alga.jpg   → se referenciará como /img/alga.jpg

Recomendaciones:
- Tamaño: 600–800 px de ancho (para tarjetas del mercado)
- Peso: < 200 KB por imagen si es posible (optimiza JPG/WebP)
- Relación de aspecto recomendada: 4:3 o 16:9
- Usa colores con buen contraste y fondo limpio

Una vez colocadas las imágenes, el código podrá referenciarlas con rutas absolutas:
- /img/coral.jpg
- /img/alga.jpg

Si luego deseas que actualice el código para renderizar estas imágenes en el Mercado, indícalo y lo integro en los ítems del mercado (marketItems) y en el componente correspondiente.
